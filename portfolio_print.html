技術スタック
Python 3.x Streamlit 1.41 Gemini 2.5 Flash Supabase (Auth + DB) gTTS Streamlit Cloud
ユーザー → Streamlit UI → シナリオ選択 → Gemini 2.5 Flash API → チャット
　　　　　　　　　　　　　　　　　　└ 評価プロンプト → JSON 解析 → Supabase DB 保存
工夫した点

① プロンプトで患者の「情報開示制限」を制御しリアルな会話を再現 ／ ② 評価を JSON 構造化出力させ SOAP 薬歴と統合パース ／ ③ Supabase で 30 分タイムアウト判定（サーバー再起動後も維持） ／ ④ 日付テンプレート（{{TODAY+3Y}}）を起動時に実日付へ動的置換
